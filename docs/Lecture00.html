可重製危機(Reproducibility crisis)

  • 再現(Replication)與重製(Reproducibility)

NAS白皮書(2019)

重製 = 任何人員使用同批資料、處理程序、分析程式碼獲得一致的分析結果。即運算可重製(computational reproducibility)

再現 = 致力回答相同問題的複數研究人員，各自獲得的資料呈現一致的結果。結果一致的程度代表可再現程度。

  • 為何是危機？
  1. 後續研究無法重現已發表的研究結果。
  2. 研究者之間無法溝通假設與研究結果的關係。
  3. 科學發展將延著累積的偏誤/偏見前進。
  4. 造假及蓄意編造的研究結果將難以被發現。

促進科學正面發展的社會規範

Merton (1942, 1973) 提出四個健全的科學知識社群賴依發展的社會規範。以下中英文說明節錄自網路文章及演講錄影，包括張邦彥(2016)以及Edward Miguel (2019)。

普遍性(Universlism)

關於真理的斷言，必須要是非關個人的。也就是說，要排除任何關於科學家種族、國籍、宗教的特殊主義考量。

共有性(Communality)

科學的實質發現是社會協作的產物，因而屬於整體社群。科學家對智慧財產權的宣稱，應只侷限於「承認」(recognition)和「尊重」(esteem)。

無私利性(Disinterestedness)

科學家應將自身的工作置放於同儕的嚴格審視之下，不可為了私利而捏造科學成果，或利用外行人的輕信建立自己的權威。

有組織的懷疑(Organized Skepticism)

科學家必須依照經驗與邏輯的標準，不妄下定論，對信念進行超然的審視。

開放透明的研究操作

  • 開放研究資料及程序素材(Open data and materials)

  • 預先註冊研究程序(Pre-registrated study design)

  • 公開同儕評審(Open peer review)
