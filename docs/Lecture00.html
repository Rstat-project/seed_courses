可重製危機(Reproducibility crisis)

  • 再現(Replication)與重製(Reproducibility)

NAS白皮書(2019)

重製 = 使用同批資料、處理程序、分析程式碼獲得一致的分析結果。即運算可重製(computational reproducibility)

再現 = 致力回答相同問題的複數研究人員，各自獲得的資料呈現一致的結果。結果一致的程度代表可再現程度。

  • 為何是危機？
  1. 後續研究無法重現已發表的研究結果。
  2. 研究者之間無法溝通假設與研究結果的關係。
  3. 科學發展將延著累積的偏誤/偏見前進。
  4. 造假及蓄意編造的研究結果將難以被發現。
