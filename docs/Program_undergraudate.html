學習準備

  1. 確認個人電腦能如影片示範安裝使用R及Rstudio。如果課程開始前無法執行，請聯絡教師安排使用Rstudio Cloud。

  2. 透過基礎 R 語法寫作示範*與示範影片系列，學習R開發環境與R基本語法。達成1.1作業驗收目標再設定個人學習進程。

* 請先下載zip檔，解壓縮後以Rstudio開啟base_R_demo.Rmd，依檔案開頭的說明，生成網頁檔瀏覽示範內容。
